# spot instance 
spot instances are unused instance in aws or azure these will provide 90% discount
Spot Instance in AWS means a temporary EC2 instance or in azure Virtual machine  that uses unused AWS capacity offered at a discount (up to 90%) compared to On-Demand pricing.

Key points (Interview-ready):

It’s cheap but interruptible — AWS can terminate it anytime when demand increases.

Best for batch jobs, CI/CD builds, big data, or fault-tolerant workloads.

Not suitable for critical or long-running applications.
